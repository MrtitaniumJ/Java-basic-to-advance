# Java Basic to Advance – Learning Workspace

Welcome! This repository is a **curriculum-style workspace** that takes you from Java fundamentals through OOP, collections/DSA, advanced Java features, system design, company-specific prep, and modern Spring-based ecosystem topics. Every folder contains focused READMEs (and many runnable `.java` examples) so you can learn progressively and practice hands-on.

## How to use this repo
- **Pick a track** based on your goal:
  - Foundations → OOP → Collections/DSA → Advanced Java → Practice/Interviews
  - System Design & Patterns → Company-Specific Prep → Modern Java Ecosystem (Spring)
- **Learn then practice**: Each concept folder has explanations, examples, and exercises/challenges.
- **Run code locally**: Open `.java` files in VS Code/IntelliJ, compile with `javac`, run with `java` (no special build needed for basic examples).
- **Use it as a reference**: Each README is a mini-guide with code snippets, pitfalls, and checklists.

## Quick navigation
- **01-Java-Fundamentals**: Basics, syntax, control flow, data types, strings, arrays, methods
- **02-OOP-Concepts**: Classes/objects, constructors, encapsulation, inheritance, polymorphism, abstraction, access modifiers
- **03-Advanced-Concepts**: Packages/import, interfaces, exception handling, multithreading, file I/O, garbage collection
- **04-Collections-and-DSA**: Java Collections Framework, data structures (lists/sets/maps/queues/stacks/trees/graphs/heaps), algorithms, patterns, complexity
- **05-Advanced-Java-Features**: Method references, streams/lambdas, file handling, networking, JDBC, memory mgmt, generics, annotations, reflection, serialization, Java 8+ features
- **06-Practice-and-Interview-Prep**: Basic → intermediate → advanced programs, interview Q&A (Core/OOP/Collections/Multithreading), projects, mock interviews
- **07-System-Design-and-Patterns**: Design patterns (creational/structural/behavioral), SOLID, system design basics, microservices architecture
- **08-Company-Specific-Preparation**: FAANG guides, Indian MNCs, coding platforms, mock interview playbooks
- **09-Modern-Java-Ecosystem**: Spring Core, Spring Boot (REST, data, security, observability, testing, deployment), cloud and performance notes

## Running code examples (CLI)
```powershell
# from the repo root, run any simple program
javac HelloWorld.java
java HelloWorld

# inside a subfolder with a .java file
cd "06-Practice-and-Interview-Prep/02-Intermediate-Programs/01-Matrix-Operations"
javac MatrixOperations.java
java MatrixOperations
```
Most examples are single-file and have a `main` method. Spring content is documentation-first (no boot projects committed) to keep the repo lightweight.

## Suggested learning paths
- **Absolute beginner → job-ready Java**: 01 → 02 → 03 → 04 (data structures/algorithms) → 05 (advanced features) → 06 (practice) → 07 (system design basics)
- **Interview fast track (DSA + core)**: 01 (refresh) → 02 → 04 (DSA + patterns) → 06 (interview Q&A + mocks)
- **Backend with Spring**: 01–03 (language) → 04 (collections/DSA basics) → 05 (streams, lambdas, JDBC) → 09 (Spring Core/Boot)
- **System design focus**: 07 (patterns + system design + microservices), then apply with mock interviews (04)
